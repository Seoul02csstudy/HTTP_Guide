### 1. 사용방법

0. 최초 git clone 진행
   
   ```
   git clone https://github.com/Seoul02csstudy/HTTP_Guide.git .
   ```

1. 작업 진행시, 본인 폴더만 수정해주세요.

2. add, commit, push 진행전 git pull을 먼저 진행해서 확인해주세요.

3. 다음 add, commit, push를 진행해 주세요.
